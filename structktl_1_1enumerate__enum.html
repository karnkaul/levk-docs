Detailed Description

template<typename Enum, Enum Begin = static_cast<Enum>(0), Enum End = Enum::eCOUNT_, typename Tr = enum_trait_linear>
struct ktl::enumerate_enum< Enum, Begin, End, Tr >

(Stateless) container for values of an Enum and its given range

Parameters
Enumenum to iterate over
Beginstart value of enum (0 by default)
Endone past the last valid value (Enum::eCOUNT_ by default)
