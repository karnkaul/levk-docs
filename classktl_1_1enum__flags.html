Detailed Description

template<typename Enum, typename Ty = std::uint32_t, typename Tr = enum_trait_linear>
class ktl::enum_flags< Enum, Ty, Tr >

Wrapper around an integral type used as bit flags, constrained to use with a particular enum.

Parameters
Enumenum to associate bit flags with
Tyunderlying type of bit flags (u32 by default)
Trtrait specifying whether enum is linear (default) or power-of-two
